INTRODUCTION
------------

NFL Sports intends to get all the NFL Teams data from the remote API.

Also need to set the custom configurations to get the data from remote API
admin/config/nflsports


REQUIREMENTS
------------

No special requirements.


RECOMMENDED MODULES
-------------------

 * Radix (https://www.drupal.org/project/radix):
   For better UI Experince need to enable this theme to use bootstrap classes

 * Components (https://www.drupal.org/project/components)
   To use Radix theme need to download and enable this module

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

Custom configuration should be must to get the Remote API Response. 
navigate this link "admin/config/nflsports" to set all the custom configurations in the form.
